First you must select the root element for tabs with jQuery selector. All direct childrens under the root work as tabs. Then you supply another jQuery selector as the first argument to the tabs constructor which selects the panes being used. This time you don't select root element - you select all the panes directly instead. This is because you don't nessesarily have a reasonable parent element for your panes.

After this JavaScript one-liner everything is pure CSS coding.
