Applications

Our thumbnail images come directly from the www.flickr.com service and the associated large images are simply the same image URL but without a "_t" substring. You are propably not using flickr, but it's very easy to make another kind of solution. Perhaps you can supply the large image's URL to the "alt" attribute of the thumbnail and get it like this:

// in the above example replace the url assignment with this line var url = $(this).attr("alt");

If you want to have more syntaxically correct pages, you might want to enclose your thumbnail images with A tags and supply the large image's url in the href attribute and maybe add some image description to the title attribute. In that case you can have the interesting opportunity to add a tooltip for scrollable items.

It is fairly easy to change the setup so that the thumbnail images will load entire HTML pages with AJAX. This could result in some good user interfaces.
