HTML coding

We have only one tooltip element and multiple triggers. Each row has a delete.png image which is configured to work as a trigger for our tooltip element. Note that we don't have any title attribute on the triggers so we can use the same content inside the tooltip for every trigger.

JavaScript code

Each tooltip is configured with this one JavaScript snippet. We have specified a tip configuration option that will select the tooltip element that is being used by all of the triggers. We also use the important lazy configuration option which will defer the tooltip initialization until the moment when a tooltip is first being used. If a page has a lot of tooltips this option will make the page load faster.
