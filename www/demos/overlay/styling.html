Overlays

CSS coding

CSS coding consists of overriding the settings of the default style. This includes customizing the background image and dimensions. Of course, you can use any CSS setting you wish to. CSS is a powerful language.

${css}

Note that this tool takes care that the background images are automatically loaded into the browser's cache so that user will see a smooth behaviour although the background image is very big. You can disable this preloading effect from the configuration.
